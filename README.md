# Personal Portfolio Website

A modern and responsive personal portfolio website built using **React**, **Vite**, **TypeScript**, and **Tailwind CSS**.  
This portfolio showcases my projects, skills, achievements, and experience with a clean UI and smooth user experience.

---

## Features

- Fast performance with Vite
- Modern and responsive UI
- Smooth animations and transitions
- Fully mobile responsive
- Component-based architecture
- Built with TypeScript for type safety
- Clean and minimal design
- Downloadable resume section
- Contact section with social links

---

## Tech Stack

- **React**
- **Vite**
- **TypeScript**
- **Tailwind CSS**
- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**

---

## Project Structure

```bash
src/
```

## Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
```

### Navigate to the project folder

```bash
cd your-repo-name
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

The app will run at:

```bash
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```
